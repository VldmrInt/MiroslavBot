# Telegram Proxy Bot

Простой Telegram-бот для равномерного распределения прокси.

## Установка

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

2. Добавьте свои прокси в файл `proxies.txt`, каждый прокси на новой строке.

3. Замените `YOUR_BOT_TOKEN` в файле `bot.py` на токен вашего бота.

## Использование

Запустите бота:
```
python bot.py
```

В Telegram отправьте боту команду `/start` или `/proxy` для получения прокси.
